# 📚 Library Management System

Un proiect Java realizat pentru cursul de Programare Orientată pe Obiect (POO) care simulează activitatea unei biblioteci.

## ✨ Caracteristici
- **Gestiune Cărți:** Adăugare, afișare și schimbarea categoriilor pentru cărți.
- **Sistem Împrumuturi:** Permite utilizatorilor să împrumute și să returneze cărți, verificând automat disponibilitatea stocului.
- **Căutare Avansată:** Căutare rapidă a titlurilor folosind prefixe.
- **Meniu Interactiv:** Interfață ușor de utilizat în consolă.

## 🛠️ Tehnologii și Concepte Utilizate
- **Limbaj:** Java
- **Colecții (Java Collections Framework):** `Map` (HashMap), `Set` (HashSet), `List` (ArrayList).
- **POO:** Clase, obiecte, constructori, metode de acces (getters/setters).

## 🚀 Cum se rulează
1. Clonează repository-ul.
2. Rulează clasa `Biblioteca.java` într-un IDE (IntelliJ/Eclipse).
3. Urmează instrucțiunile din meniul afișat în consolă.
